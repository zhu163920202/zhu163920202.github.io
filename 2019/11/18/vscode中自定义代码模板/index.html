vscode中自定义代码模板

  1. 打开setting -> user snippets
  2. 选择需要定义的文件，如：html.json
  3. 配置json
  4. 在html文件中输入vue-html+ enter即可生成自定义的模板
  5. 注："prefix":相当于自定义快捷键；"body"即模板内容；"description"是模板的描述内容;有些属性的双引号冲突用\转义符即可
